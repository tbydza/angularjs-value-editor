patternDescriptionTooltip

  1. - directive in module angularjs-value-editor

This directive adds tooltip on input, if valueEditor has defined non-empty localization named patternDescription.

Dependencies

  • ^^kpValueEditor

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    	<pattern-description-tooltip>
	...
</pattern-description-tooltip>
  • as attribute: 
    <ANY data-pattern-description-tooltip>
	...
</ANY>

kpUniversalForm

  1. - component in module angularjs-value-editor

Component for generating forms by definition passed via formSettings attribute.

Usage

<kp-universal-form
    form-settings="KpUniversalFormComponentSettings"
    [name="string"]
    [form-controller=""]
    [submit-function=""]
    [labels-width="ObjectValueEditorLabelsWidth"]>
</kp-universal-form>

Arguments

Param Type Details
formSettings KpUniversalFormComponentSettings

Definition of form content.
name
(optional)
 String

Name of the form. Due to internal reason, it must be in accordance with ^[a-zA-Z0-9._]*$ regexp.
formController
(optional)
 function(IFormController)=

Connecting to controller.

Param Type Details
$formController
(optional)
 function(IFormController)=

Exposed form controller.
submitFunction
(optional)
 function(Event)=

Function called on submit form.

Param Type Details
$event
(optional)
 Event

Submit event.
labelsWidth
(optional)
 ObjectValueEditorLabelsWidth

See ObjectValueEditorOptions. Default value is 2.

Example

<main class="container" ng-controller="controller as $ctrl">
     <kp-universal-form ng-model="model" name="myForm" form-settings="$ctrl.formSettings" form-controller="formController = $formController"></kp-universal-form>
     <div>Model:</div>
     <pre ng-bind="model | json"></pre>
     <div>Form controller:</div>
     <pre ng-bind="formController | json"></pre>
</main>
